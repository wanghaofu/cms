SetFillColor

SetFillColor(int r [, int g, int b])

版本

1.3

内容

这个颜色的定义可以用在所在填满操作上(filled rectangles(填满平衡四边形)和cell backgrounds(单元的背景))。可以使用RGB色彩或灰阶模式来定义。这个功能可以在一页之前建立，而且那个色彩值会保留在一页到另一页。

参数

r
若果g 和 b有提供数值，这个功能就是(红色的色彩)。如果没有提供任何数据在g 和 b，这个功能就是(灰阶的色彩). 数值为 0 至 255 之间。
g
绿色的色彩 (0 至 255 之间).
b
蓝色的色彩 (0 至 255 之间).

参见

SetDrawColor(), SetTextColor(), Rect(), Cell(), MultiCell().
