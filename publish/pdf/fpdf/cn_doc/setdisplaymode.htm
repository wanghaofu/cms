SetDisplayMode

SetDisplayMode(mixed zoom [, string layout])

版本

1.2

内容

定义文件在显示出来的模式。可以设定(缩小放大)的级数：使到显示出来的页面为全画面、窗口的全宽度、使用真正的大小、安比例(缩小放大)或使用默认值(已Acrobat作为标准定义)。 还可以具体规划页面：只有一页、连续显示、两个字段或浏览程序的默认值。 默认值：文件使用全宽度和连续显示模式。

参数

zoom
使用(缩小放大)功能。这个功能可以使用已下的数值：
  • fullpage: 使到显示出来的页面为全画面
  • fullwidth: 窗口的全宽度(用窗口的最大宽度)
  • real: 使用真正的大小 (相等于 100% (缩小放大))
  • default: 使用浏览程序的默认值
或用数值来定义(缩小放大)参数
layout
页面规划。这个功能可以使用已下的数值：
  • single: 只有一页
  • continuous: 连续显示
  • two: 显示两页在两个字段上
  • default: 使用浏览程序的默认值
默认值：连续显示
