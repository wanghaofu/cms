Header

Header()

版本

1.0

内容

这个功能是用来产生页首。AddPage() 和 Close() 会自动呼叫这个功能，但程序不会直接呼叫这个功能。这个结果在 FPDF 里是空白的，若果你想进行特殊处理，你要透过(subclass)去撤销这个功能。

例子

class PDF extends FPDF
{
function Header()
{
    //Select Arial bold 15
    $this->SetFont('Arial','B',15);
    //Move to the right
    $this->Cell(80);
    //Framed title
    $this->Cell(30,10,'Title',1,0,'C');
    //Line break
    $this->Ln(20);
}
}
