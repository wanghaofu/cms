Image

Image(string file, float x, float y [, float w [, float h [, string type [, mixed link]]]])

版本

1.1

内容

把一个图像放在页里面。图像的左上角作为定点坐标。尺寸定义可以使用不同的方法：
  • 清楚的宽度和长度(自行定义尺寸单位)。
  • 详尽的尺寸：为了保持图像基本尺寸比例，它会自动计算尺寸，
  • 是没有需要定义详尽尺寸。在这个事例，图象定为每寸72点(72 dpi)。
支持 JPEG 和 PNG 格式。

JPEG，所有支持模式：
  • gray scales
  • true colors (24 bits)
  • CMYK (32 bits)
PNG，所有支持模式：
  • gray scales 最大支持 8 bits (256 levels)
  • indexed colors
  • true colors (24 bits)
但不支援：
  • Interlacing (交错式)
  • Alpha channel
它会很重视透明色的定义。(但只限于 Acrobat 4 或已上版本). 图像格式可以自行定义或由图像文件名称后面的扩展名来推论出图像的格式。 它可以加入连结在图像上。 备注：若果一个图像会用上几次，只需要一个图像档案便足够。

参数

file
图像文件名称
x
左上角的横坐标(x)
y
左上角的纵坐标(y)
w
在页面上图像的宽度。如果没有定义或定义为0，它会自动计算。
h
在页面上图像的高度。如果没有定义或定义为0，它会自动计算。
type
图像格式。格式值为：JPG、JPEG、PNG。如果没有定义图像格式，会由图像文件名称后面的扩展名来推论出图像的格式。
link
URL 或 identifier 由 AddLink() 处理。
