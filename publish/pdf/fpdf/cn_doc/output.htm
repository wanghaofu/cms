Output

string Output([string name [, string dest]])

版本

1.0

内容

送出文件到指定目标，例如：字符串，本地档案或浏览器。 在最后一个步骤，可能会用到外挂程序("若存在")或强行则行下载("另存" 对话盒)。
若需要终断文件，Output()这个功能会首先则行Close()。

参数

name
文件名称。如果没有提供文件名称，当文件存送到浏览器(dest I)时，文件名称会自动定义为doc.pdf。
dest
送出文件的目标位置。可用下面任何一个值：
  • I：送出档案到浏览器上。如果有外挂程序存在的话，也可同时使用。使用者可透过连结去另存PDF档案，文件名称会由 (name 参数)定义。
  • D：存送到浏览器，并且强行下载存档，文件名称会由 (name 参数)定义。
  • F：储存到本机档案，文件名称会由 (name 参数)定义。
  • S：透过字符串反回文件，文件名称不需要使用。假如，dest的参数没有定义，但已经定义了文件名称，那 dest的参数会自动定义为F。若果什么参数都没有定义的话，那 dest的参数会自动定义为I。

    注意：为了兼容之前的版本，boolean 值是也会接受的。(错误为F、正确为D)

参见

Close().
